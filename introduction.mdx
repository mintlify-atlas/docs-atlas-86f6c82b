Huayu Buddy is a Mandarin Chinese learning app that puts you in a live spoken conversation with an AI tutor. You speak, the tutor responds in natural Mandarin with audio, Hanzi, and Pinyin — and gently corrects your mistakes along the way.

## Architecture overview

Huayu Buddy is composed of three main layers:

**React UI (`/src`)**
The browser-side interface handles conversation flow, the HSK vocabulary browser, and the 3D avatar rendered with Three.js and `@react-three/fiber`. A separate Realtime panel manages WebRTC-based streaming chat.

**Express API (`/server`)**
A Node.js/Express server exposes the AI endpoints that the UI calls:

| Endpoint | Purpose |
|---|---|
| `POST /api/chat` | Standard chat completion (GPT) |
| `POST /api/tts` | Text-to-speech audio generation |
| `POST /api/stt` | Speech-to-text transcription (Whisper) |
| `POST /api/realtime-session` | Issues ephemeral Realtime API tokens |
| `GET /api/hsk` | Returns HSK word list filtered by level |
| `GET /api/health` | OpenAI connectivity health check |

**Electron wrapper (`/electron`)**
Bundles the React build and Express server into a single desktop executable. Manages the application window, system tray, and packaging for Windows.
