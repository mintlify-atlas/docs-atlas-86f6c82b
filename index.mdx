## What is Huayu Buddy?

Huayu Buddy is a full-stack Mandarin Chinese learning application that puts an AI tutor right on your desktop or in your browser. Speak in Mandarin or English, and the tutor responds with natural Mandarin audio, Hanzi characters, Pinyin romanization, and optional English translations — all delivered through a lip-synced 3D avatar.

<CardGroup cols={2}>

  </Card>
  <Card title="3D tutor avatar" icon="user">
    A head-only Three.js avatar with lip-sync visemes and natural blinking reacts as the tutor speaks.
  </Card>

## How it works

Huayu Buddy is composed of three main layers that work together:

<Steps>
  <Step title="React UI (src/)">
    The browser-side application handles the conversation UI, HSK vocabulary browser, 3D avatar rendering via Three.js, and the optional WebRTC realtime panel.
  </Step>
  <Step title="Express API (server/)">
    A lightweight Node.js server proxies requests to the OpenAI API — providing chat completion, text-to-speech, speech-to-text, and ephemeral Realtime session tokens.
  </Step>
  <Step title="Electron wrapper (electron/)">
    For desktop use, Electron bundles the React app and Express server into a single portable executable with a system tray and no external dependencies.
  </Step>
</Steps>
